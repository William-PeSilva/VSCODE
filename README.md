# TREINAMENTO
Estou realizando alguns treinamentos para aprimorar meus estudos, com isso fiz um pequeno programa com fundamentos simples utilizando o VSCode voltado a linguagem Python.
Nele fiz um programa que pedisse o nome do usuário, em seguida mostrando a mensagem Olá, NOME. Após isso, utilizei código para pedir a idade e em seguida utilizando if/else se era maior ou menor de idade. Por último solicitei que o usuário me passasse um número aleatório e utilizando o if/else para verificar se o número seria positivo ou negativo.
Por fim e último, acabei juntando todo os códigos com as respostas digitadas pelo usuário e juntei formando uma frase com as informações repassadas.
